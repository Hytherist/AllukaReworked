# AllukaReworked

# Slash Commands:

# setStatus - Takes three parameters `<string status>, <string type-of-activity>, <string activity>`
Users can set the status, activity type, and specific activity of the bot.

# event - Takes five parameters `<string name>, <int hour>, <int minute>, <string time-of-day>, <string location>`
Users can create an embed with the following information: Name of event, time, time of day, and location.
